Konteks & Masalah

Mengapa Prediksi Digital?

Berat ikan merupakan parameter krusial dalam menentukan nilai ekonomi dan efisiensi logistik. Metode konvensional memerlukan penimbangan fisik yang memakan waktu dan berisiko menyebabkan stres pada ikan (handling stress).

Solusi Machine Learning:

Memanfaatkan variabel dimensi fisik (panjang, tinggi, lebar) untuk mengestimasi berat secara instan tanpa kontak fisik langsung.

Tujuan Utama

  • Membangun model regresi akurasi tinggi.
  • Komparasi efisiensi Linear vs Forest.
  • Identifikasi fitur fisik paling dominan.
  • Validasi hasil Modeling vs EDA.
Dataset

Fish.csv

Berisi 159 entri data dengan 7 variabel fisik ikan lintas spesies.

Target Variabel Weight
Fitur Independen Species, L1-L3, H, W

Data Preprocessing

Kualitas data menentukan kualitas prediksi. Tahapan pembersihan meliputi:

  • • Filtering: Menghapus baris dengan Weight ≤ 0.
  • • Encoding: Transformasi label spesies ke numerik.
  • • Scaling: Normalisasi fitur via StandardScaler.
  • • Splitting: Pembagian data (80% Train / 20% Test).

Modeling & Performance

Hasil Evaluasi Komparatif

Algorithm MAE RMSE R² Score
Linear Regression 72.41 104.35 0.92
Random Forest 43.87 66.12 0.97

Analisis: Model Random Forest berhasil menurunkan error (MAE) sebesar hampir 40% dibanding Linear Regression. Akurasi R² mencapai 0.97, artinya model mampu menjelaskan 97% variansi data berat ikan.

Kenapa Random Forest?

Algoritma berbasis pohon (Tree-based) ini jauh lebih tangguh dalam menangani hubungan non-linear dan korelasi antar fitur yang kompleks antar spesies ikan yang berbeda-beda.
